1. Playback:
	a. Launch Sahi Editor by clicking on 'Editor' link on the 'Sahi Dashboard'.
	
	b. Search for 'Automation.dd.csv' file in editor navigator and open.
	
	c. Click on 'Playback', Select 'Browser', Enter 'Start url' and click on 'Run'.
	
	d. Once playback is complete, Click on 'Logs' link to view the reports.

2. Types of Playback:
	a. Playback via Editor
			http://localhost:9999/_s_/docs/using-sahi/sahi-script-editor.html#Playback
			Or, https://resources.sahipro.com/docs/using-sahi/sahi-script-editor.html#Playback
	
	b. Playback from command line
	        http://localhost:9999/_s_/docs/using-sahi/playback-commandline.html
			Or, https://resources.sahipro.com/docs/using-sahi/playback-commandline.html
	
	c. Playback via Ant
			http://localhost:9999/_s_/docs/using-sahi/playback-via-ant.html
